小结

综上可以了解到，Android应用的AndroidManifest.xml文件是在PMS启动时，扫描Apk的过程中通过PackageParser对Apk进行解析处理。

最后，回到开始的问题，我们需要处理的是对应应用的方向，所以在解析Activity的时候，解析到a.info.screenOrientation的时候，根据应用的packageName进行对screenOrientation赋值即可。
